# UdangPro — Sistem Pengurusan Ladang Udang

🌐 **Website:** https://nephicrosis-boop.github.io/UdangPro

Sistem pengurusan ladang udang dengan sync cloud. Data disimpan di Supabase dan boleh diakses dari mana-mana device.

## ✨ Ciri-ciri
- 11 modul: Dashboard, Kolam, Air, Benih, Makanan, Pertumbuhan, Kesihatan, Tuaian, Kewangan, Pengguna, Laporan
- CRUD penuh untuk semua data
- Sync automatik ke cloud (Supabase)
- Offline mode — data kekal guna localStorage
- PWA — boleh Add to Home Screen kat phone


## 🛠 Tech Stack
- React 18 (single-file, CDN)
- Tailwind CSS
- Recharts
- Supabase (database + API)
- GitHub Pages (hosting)
